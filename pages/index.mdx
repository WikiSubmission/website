## Contribute

For now, content contributions are possible only supported via GitHub pull requests. This requires some familiarity with markdown & Git software. Please note:

* The code for this website is fully [open-source](https://github.com/wikisubmission/website)
* All additions or changes are [public](https://github.com/wikisubmission/website/commits)

To be added to the repository, send us an [email](mailto:contact@wikisubmission.org) including your GitHub email and how you wish to contribute. If you're on Discord, send a [DM](https://discord.com/channels/@me/911268375194386552) instead. Any updates to the repository take around 5-10 minutes to reflect on the actual website.
