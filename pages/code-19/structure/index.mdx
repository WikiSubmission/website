Every book in existence has a mathematical structure to it. A simple, universal example would be the number of **pages** it contains. Going deeper, most books are further divided into **chapters**. Each chapter typically has a certain number of **paragraphs**. Each paragraph has a certain number of **words**. Each word has a certain number of **letters**. Each letter... well, that's the smallest unit for a book.
